Ladder Management

Edit ladder properties

Ladder administrators can edit ladder properties by switching to Admin mode from the main menu, and selecting “Ladder Settings”. The ladder name, club name, region, ranking type and privacy status can be edited.

Deleting a ladder

There is currently no way to delete a ladder, however, as a ladder administrator, you can remove all competitors from the ladder and, finally, remove your own competitor. This will remove the ladder from all players' ladder menus. The ladder will still be listed if you search for it when trying to add a ladder, but will automatically be deleted completely at a later date.

Adding competitors

In admin mode, ladder administrators can add competitors to their ladders in several ways, depending on how they run the ladders.

  • Publicly accessible ladders:
    If a ladder is configured to be public, players can join the ladder by searching for it in their region. Public ladders allow new players to find opponents in their area.
  • Sharing a ladder code:
    Whether a ladder is public or private, administrators can share the ladder code to allow players to easily join the ladder. When players join a ladder using a shared ladder code, a new competitor will be created for each player that joins the ladder.
  • Sharing a competitor code:
    If ladder administrators would prefer to manually add competitors to their ladder, they can do so, and then share the unique competitor codes for each competitor with individual players. For new players to be able to join the ladder, they will need to first get a competitor code from the administrator.

Sharing a ladder code

Ladder administrators can view and share their ladder codes by switching to Admin mode and editing their ladder. They can then click on the share button next to the ladder code to share it via social media, email, etc. They can also display their ladder code at their local club, to allow club members to join the ladder.

Manually adding competitors

Ladder administrators can manually add new competitors to their ladders by pressing the “+” button that is shown on the ladder tab in Admin mode. They will be prompted to enter the competitor's display name, and optionally add an avatar for the competitor. They can then share the newly created competitor's code with a player (or players for doubles/team ladders), to allow the player to join the ladder. To do this they can press the “Add player” option to share the code with the player/players to link to the competitor. To share the code at another time, they can select the competitor in the ladder (in admin mode) to edit the competitor, and share the competitor code as described above. After the maximum number of players have linked to the competitor, ladder administrators will no longer be able to share their codes.

Editing competitors

To edit an existing competitor, select the competitor on the Ladder tab, in Admin mode, to view the “Edit Competitor” page. Administrators can edit the competitor's “Display name” or avatar.

Removing competitors

To remove a competitor from the ladder, select the competitor on the Ladder tab, in Admin mode, and press the “Remove” button on the “Edit Competitor” page.

Editing players

To edit a player, ladder administrators must first edit a competitor that is linked to the player to see the players that are linked to the competitor. After selecting the player in the list, they will be able to edit the player's details in the “Edit Player” page.

Administrators can edit a player's name, surname or email address; reset their password; or grant them administrative access to the ladder.

Resetting a player's password

Player's can reset their own passwords from the login page. However, if they no longer have access to the email address they used to sign up in the app, administrators can reset their password. To reset a player's password, edit the player and press the “Reset Password” button. Enter a temporary password and share it with the player. They will be able to use it to log in to the app one time, after which they will be prompted to enter a new password.

Granting players administrative access

It is advisable to have more than one administrator for each ladder. To grant a player administrative access to a ladder, edit the player and switch on the “Ladder administrator” toggle switch.

Adding results

Ladder administrators can capture results for competitors who aren't linked to any app users, as well as for tournament results. Results that are captured by administrators don't need to be confirmed by the players. After capturing a result, a push notification will be sent to all players linked to the 2 competitors.

To capture a result of a match played between 2 competitors, switch to Admin mode and press the “+” button on the results tab. Select the first competitor from the list of ladder competitors that is presented. (For consistency, it is recommended to select the winner first, but this is not required) Then select the other competitor. Thereafter, you will be able to enter the date and time of the match, as well as the result. You can either enter the overall match result (e.g., 3 - 0), or the individual game scores (e.g., 15-8, 15-7, 16-14).

Tournament results

Tournament results are results that were played as part of an organized tournament. Only ladder administrators can capture tournament results. Tournament results are weighted more heavily than regular results, and will result in bigger changes to competitors' ranking points. To capture a tournament result, switch on the tournament result toggle switch.

Editing results

Ladder administrators can make changes to results that have been captured by themselves, or other players, even after the result has been confirmed. To edit a result simply select it in the Results tab in Admin mode, make the required changes, and save the result.

Deleting results

To delete a result that was incorrectly captured, select it in the Results tab in Admin mode, and press the “Delete” button.
