Slappy Login Page

When starting the app for the first time, players will see the login page. From this page, players can log into the app, create a new account or create a new ladder.

Creating a user account

Before creating an account, new users must first create a new ladder or join an existing one. They can join using a code that they have received from their club or another ladder member, or by searching for a public ladder in their area.

Using a code

If a new player has received a ladder code or a competitor code, they can use it to join a ladder by pushing the “Create account/join ladder” button and then selecting “Enter ladder/competitor code”. After entering their code, they can click the “Submit code” button. Thereafter, they can complete the user registration process.

Creating a ladder or searching for a public ladder

If a player would like to create a new ladder or join a ladder without a ladder code, they can do this by pushing the “Create account/join ladder” button and then selecting “Search/create ladder” button. They can then select their country and region to list all available ladders for their region. If they find the ladder they are searching for, and it is public, they can join the ladder by selecting it. If they find the ladder, but it is marked as private, they will need to contact the club or other ladder members to request a code to join the ladder. If the ladder they are looking for doesn't exist, they can press the “+” button to create a new ladder. If no ladders are found in a region, the player will automatically be prompted to create a new ladder.

Creating a new ladder

If, after searching for a ladder, a user can't find it, they can create a new ladder by pressing the “+” button. The user will need to enter a unique ladder name for their region, and optionally select an image for the ladder and their club name.

Registering a user

After selecting a ladder to join, or creating a ladder, the user must enter their name, surname, email address and password, to register themselves on the system. A competitor, linked to the ladder, will automatically be created for the player when they register. Thereafter, the user will be able to login to the app using the email address and password they have supplied.

They can also optionally add an avatar that will be used to identify them on the ladder. Although adding an avatar is optional, it is highly recommended to provide a head and shoulders photograph, to be clearly identifiable, especially to new players on the ladder..

Logging in

After a player has registered themselves, they can login to the app using their email address and password.

Remember password

When logging in, players can choose to remember their password. If this option is selected, they will not need to log in again in future, unless they choose to log out from the main menu.

Recover forgotten password

If a player forgets their password, they can recover it by entering their email address and selecting the “Forgot password?” link. After confirming that they want to reset their password, they will be sent an email containing a verification code, which they can enter on the “Reset Password” page, to create a new password.
