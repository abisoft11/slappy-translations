Slappy Ladders

The Slappy Sports Ladder app is used to manage sports ladders. New ladders can be created when a new user is registering themselves. The user creating the ladder will automatically become a ladder administrator for that ladder.

Creating or joining ladders

While registering a new user, it is necessary for that user to join an existing ladder or to create a new ladder. After logging in to the app, users can create or join other ladders by touching their ladder icon and selecting the “Add Ladder” menu option. After selecting this option, the process is the same as the initial process for creating or joining a ladder.

Ladder types

When ladders are created, their type must be configured by the administrator. Ladder types are defined by the following options:

  • Sport: The sport that the ladder is used for. The following sports are currently supported:
    • Badminton
    • Darts
    • Foosball
    • Handball (American or Gaelic)
    • Paddleball (one, three or four wall)
    • Padel
    • Pelota
    • Pickleball
    • Racquetball
    • Real Tennis
    • Squash
    • Table Tennis
    • Tennis
    • Touchtennis
    • Volleyball
  • Format: The number of players for each competitor:
    • Singles
    • Doubles
    • Mixed Doubles

Ranking type

The ranking type used to order competitors on the ladder must also be configured for new ladders. The ranking type can be one of the following:

  • ELO: Points-based ranking system based on the chess ELO system
  • Leapfrog: If the lower ranked competitor wins a match, they “leapfrog” the higher ranked competitor on the ladder.

Multiple ladders

Basic and premium subscribers can belong to multiple ladders. Touching the ladder icon will present a list of ladders that the player belongs to. Selecting a ladder from the list will show the selected ladder on the ladder tab. It is also possible to view results for the selected ladder on the results tab.

Results are not linked to ladders. This means that you can belong to multiple ladders with some of the same competitors in both ladders. If you capture a result against an opponent that belongs to more than one of your ladders, the result will be captured for all of the shared ladders. The points change will most likely be different in each of the ladders.

Public and private ladders

When creating or editing a ladder, it is possible to make them either public or private. Anyone can search for and join a public ladder, however, players can only join a private ladder if they have the ladder's code or a competitor code for the ladder.

Ladder management

Ladder administrators can edit ladder properties as well as performing other ladder management tasks.
