Slappy Ladders

The Slappy Sports Ladder app is used to manage sports ladders. New ladders can be created when a new player is being registered. The player registering the ladder will automatically become a ladder administrator for that ladder.

Joining a ladder

When registering a new player, it is necessary for that player to join an existing ladder or to create a new ladder. There are 3 ways to find an existing ladder:

  • Ladder code: If the player has been given an 8-digit code for the ladder, this can be entered to quickly find the ladder. Ladder codes can be sent/given to new players by ladder administrators or they could be displayed at the sports club where the ladder is hosted.
  • Competitor code: If the player has been given a 7-digit competitor code, this can be entered to quickly join the ladder with a preconfigured competitor. Competitor codes can be sent/given to new players by ladder administrators.
  • Location Search: New players can search for a ladder in their area, by first selecting their country and region, and then choosing from existing ladders in that region, by name.

Adding a new ladder

If, after searching for a ladder, a player can't find their ladder, they can create a new ladder by pressing the “+” button. The player will need to enter a unique ladder name for their region, and optionally select an image for the ladder and their club name.

If no ladders are found in a region, the player will automatically be prompted to add a new ladder.

After adding the ladder, the player must immediately register themselves on the system, and they will become an administrator for that ladder. Thereafter, they can either add new competitors themselves, or share their ladder code with players who belong to the ladder.

Ladder types

When ladders are created, their type must be configured by the administrator. Ladder types are defined by the following options:

  • Sport: The sport that the ladder is used for. The following sports are currently supported:
    • Badminton
    • Foosball
    • Padel
    • Racquetball
    • Squash
    • Table Tennis
    • Tennis
  • Format: The number of players for each competitor:
    • Singles
    • Doubles
    • Mixed Doubles

Ranking type

The ranking type used to order competitors on the ladder must also be configured for new ladders. The ranking type can be one of the following:

  • ELO: Points-based ranking system based on the chess ELO system
  • Leapfrog: If the lower ranked competitor wins a match, they “leapfrog” the higher ranked competitor on the ladder.

Multiple ladders

Basic and premium subscribers can belong to multiple ladders. Touching the ladder icon will present a list of ladders that the player belongs to. Selecting a ladder from the list will show the selected ladder on the ladders tab. It is also possible to view results for the selected ladder on the results tab.

Results are not linked to ladders. This means that you can belong to multiple ladders with some of the same competitors in both ladders. If you capture a result with an opponent that belongs to more than one of your ladders, the result will be captured for all of the shared ladders. The points change will most likely be different in each of the ladders.

Public and private ladders

When creating or editing a ladder, it is possible to make them either public or private. Anyone can search for and join a public ladder, however, players can only join a private ladder if they have the ladder's code or a competitor code for the ladder.

Sharing a ladder code

Ladder administrators can view and share their ladder codes by switching to Admin mode and editing their ladder. They can then click on the share button next to the ladder code to share it via social media, email, etc. They can also display their ladder code at their local club, to allow club members to join the ladder.
