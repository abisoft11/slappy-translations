Slappy Spieler-Management

Spieler

Spieler werden von der App mittels ihrer E-Mail-Adresse eindeutig identifiziert. Nach der Installation der App, müssen Spieler ein Konto eröffnen und sich mit den Ranglisten verbinden, welchen sie angehören wollen. Dazu können Spieler entweder neue Ranglisten gründen oder bestehenden beitreten. Das können sie tun, indem sie den zur Rangliste passenden Ranglisten-Code verwenden, indem sie die Rangliste in ihrer Region suchen oder indem sie den individuellen Teilnehmer-Code benutzen. Danach muss der Spieler seine persönlichen Angaben auf der Spieler-Seite eingeben.

Teilnehmer

Teilnehmer sind Spieler oder Teams, die einer Rangliste angehören. Wenn die Rangliste für Doppelspieler ist, kann ein Teilnehmer zwei Spieler beinhalten. Standardmässig werden bei Teilnehmer die Avatare der Spieler angezeigt. Falls gewünscht, kann man jedoch einen anderen Avatar als Teilnehmer wählen. Teilnehmer können in jeder Rangliste, in der sie teilnehmen, verschiedene Namen und Avatare verwenden.

Als Spieler registrieren

Nachdem eine Rangliste gewählt wurde, muss der Spieler seinen Namen, Vornamen, E-Mail-Adresse und Passwort eingeben. Optional kann er auch einen Avatar hochladen um sich in der Rangliste besser auszuweisen. Das wird sehr empfohlen.

Spieler-Details aktualisieren

Nachdem sich ein Spieler bei der App anmeldet, kann er die Details (Name, E-Mail, Passwort, Avatar) zu seiner Person bearbeiten. Der Spieler, der eine Rangliste gegründet hat, wird automatisch die Rolle des Ranglisten-Administratoren zuteil. Ranglisten-Administratoren können andere Spieler ebenfalls zu Ranglisten-Administratoren ihrer Rangliste ernennen. Es ist zu empfehlen pro Rangliste mehr als einen Ranglisten-Administratoren zu haben.

Um andere zu Ranglisten-Administratoren zu ernennen, muss der Administrator im Hauptmenu in den "Admin"-Modus wechseln. In diesem Modus kann er den gewünschten Teilnehmer auswählen und den dazugehörigen Spieler zum Administratoren ernennen.

Administratoren können lediglich Teilnehmer-Details bearbeiten, nicht aber Spieler-Details.

Teilnehmer hinzufügen

Ranglisten-Administratoren können im Voraus Teilnehmer (individuelle Spieler oder Teams) ihrer Rangliste zuordnen und dann die betroffenen Teilnehmer mittels individuellen generierten Teilnehmer-Codes zur Rangliste einladen. Nachdem sie die App runtergeladen haben, können die Spieler diesen Code eingeben und der Rangliste beitreten. Bei Doppelranglisten können 2 Spieler als ein Teilnehmer auftreten.

Teilnehmer, welche von einem Ranglisten-Administratoren hinzugefügt wurden, können ihre persönlichen Details (Anzeigenamen, Avatar) nach der Anmeldung bei der App aktualisieren.

Wenn Spieler Doppelranglisten beitreten, können Sie den Teilnehmer-Code mit ihrem Partner teilen. Wenn der Partner sich mit dem Code bei der App anmeldet, wird er automatisch als Partner identifiziert und hinzugefügt. Nur 2 Spieler können einem Doppel-Teilnehmer angehören.
