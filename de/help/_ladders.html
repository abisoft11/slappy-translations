Slappy Ranglisten

Die App Slappy Sportranglisten dient dazu, Sport-Ranglisten zu verwalten. Neue Ranglisten können angelegt werden, wenn sich ein neuer Spieler registriert. Der Spieler, der eine Rangliste anlegt, wird automatisch zum Ranglisten-Administratoren.

Eine Rangliste erstellen oder einer Rangliste beitreten

Wenn sich ein neuer Spieler registriert, sollte er einer bestehenden Rangliste beitreten oder eine neue Rangliste erstellen. Nach dem Login können Benutzer einer Rangliste beitreten oder eine Rangliste erstellen, indem sie das Ranglisten-Icon berühren und das Menu „Rangliste erstellen“ wählen. Nachdem diese Option gewählt wurde, ist der Ablauf um eine Rangliste zu ertellen oder einer Rangliste beizutreten der gleiche wie nach der erstmaligen Registration.

Ranglisten-Arten

Wenn eine Rangliste kreiert wurde, muss der Ranglisten-Administrator diese konfigurieren. Ranglisten-Arten sind wie folgt definiert:

  • Sport: Der Sport für den die Rangliste verwendet wird. Die folgenden Sportarten werden derzeit unterstützt:
    • Badminton
      • +++
    • Cornhole
    • Darts
    • Tischfussball
    • Handball (amerikanisch oder gälisch)
    • Paddleball (eine, drei oder vier Wände)
    • Padel-Tennis
    • Pelota
    • PickleBall
    • Racquetball
    • Jeu de Paume (Echtes Tennis bzw. Real Tennis)
    • Squash
    • Tischtennis
    • Tennis
    • Touchtennis
    • Volleyball
  • Format: Die Anzahl Spieler pro Teilnehmer/Team:
    • Einzel
    • Doppel
    • Gemischtes Doppel

Ranglisten-Methoden

Die Ranglisten-Methode, welche die Spieler in der Rangliste einordnet muss konfiguriert werden. Es stehen folgende Ranglisten-Methoden zur Auswahl:

  • ELO: Punkte-Wertungssystem, basierend auf dem Schach-ELO-System.
  • Leapfrog: Wenn der schlechter klassierte Spieler einen Match gewinnt, setzt er sich in der Rangliste an die Stelle des höher klassierten Spielers, gegen den er gewonnen hat.

Mehrere Ranglisten

Basic- und Premium-Abonnenten können an mehreren Ranglisten teilnehmen. Wenn man das Ranglisten-Icon drückt, erhält man eine Übersicht aller Ranglisten, an denen man teilnimmt. Wählt man aus dieser Übersicht eine Rangliste, wechselt der die Reiter Rangliste sogleich auf die gewählte Rangliste. Es ist auch möglich die Resultate der gewählten Rangliste im Reiter Resultate zu sehen.

Resultate sind nicht an Ranglisten gebunden. Für Spieler, die gleichzeitig in mehreren gleichen Rangliste vertreten sind, gelten eingetragene Resultate für alle Ranglisten, an denen beide Spieler teilnehmen. Die Punkteveränderung kann jedoch von Rangliste zu Rangliste variieren.

Öffentliche und private Ranglisten

Wenn eine Rangliste erstellt wird, kann diese entweder öffentlich oder privat konfiguriert werden. Jeder kann öffentliche Ranglisten suchen und beitreten. Um einer privaten Rangliste beizutreten, benötigt man deren Ranglisten-Code oder den individuellen Teilnehmer-Code.

Ranglisten-Management

Ranglisten-Administratoren können Ranglisten-Einstellungen editieren.
