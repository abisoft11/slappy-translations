Slappy Ranglisten

Die App Slappy Sportranglisten dient dazu, Sport-Ranglisten zu verwalten. Neue Ranglisten können angelegt werden, wenn sich ein neuer Spieler registriert. Der Spieler, der eine Rangliste anlegt, wird automatisch zum Ranglisten-Administratoren.

Einer Rangliste beitreten

Wenn sich ein neuer Spieler registriert, sollte er einer bestehenden Rangliste beitreten oder eine neue Rangliste kreieren. Es gibt drei Möglichkeiten, eine bestehende Rangliste zu finden:

  • Ranglisten-Code: Wenn dem Spieler ein 8-stelliger Ranglisten-Code für eine Rangliste gegeben wurde, kann er diesen eingeben um die Rangliste zu finden. Ranglisten-Codes können vom Ranglisten-Administratoren an neue Spieler verschickt werden oder man verteilt den Code z.B. in einem Club, der die Rangliste führt.
  • Teilnehmer-Code: Wenn dem Spieler ein 7-stelliger Teilnehmer-Cod gegeben wurde, kann er diesen eingeben um der Rangliste beizutreten. Der Teilnehmer wurde zuvor bereits vom Ranglisten-Administratoren erfasst. Dieser kann die Teilnehmer-Codes an die erfassten Spieler weitergeben.
  • Suche nach Standort: Neue Spieler können nach Ranglisten in ihrer Umgebung suchen, indem sie zuerst ihr Land und ihre Region angeben und dann in den Suchresultaten die gewünschte Rangliste wählen.

Eine neue Rangliste anlegen

Wenn der Spieler keine Rangliste finden konnte, kann er eine neue Rangliste anlegen, indem er den "+"-Button drückt. Dazu muss er der Rangliste einen Namen geben, der in seiner Region noch nicht verwendet wird. Optional kann er für die Rangliste ein Bild hochladen und mit seinem Clubnamen versehen.

Wenn in einer Region keine Rangliste gefunden werden konnte, wird der Spieler automatisch dazu aufgefordert, eine neue Rangliste anzulegen.

Nachdem er die Rangliste angelegt hat, muss sich der Spieler sogleich registrieren, womit er zum Ranglisten-Administratoren ernennt wird. Anschliessend kann man entweder neue Spieler hinzufügen oder den Ranglisten-Code mit Spielern teilen, die der Ranglisten angehören sollen.

Ranglisten-Arten

Wenn eine Rangliste kreiert wurde, muss der Ranglisten-Administrator diese konfigurieren. Ranglisten-Arten sind wie folgt definiert:

  • Sport: Der Sport für den die Rangliste verwendet wird. Die folgenden Sportarten werden derzeit unterstützt:
    • Badminton
    • Padel-Tennis
    • Racquetball
    • Squash
    • Tischtennis
    • Tennis
  • Format: Die Anzahl Spieler pro Teilnehmer/Team:
    • Einzel
    • Doppel
    • Gemischtes Doppel

Ranglisten-Methoden

Die Ranglisten-Methode, welche die Spieler in der Rangliste einordnet muss konfiguriert werden. Es stehen folgende Ranglisten-Methoden zur Auswahl:

  • ELO: Punkte-Wertungssystem, basierend auf dem Schach-ELO-System.
  • Leapfrog: Wenn der schlechter klassierte Spieler einen Match gewinnt, setzt er sich in der Rangliste an die Stelle des höher klassierten Spielers, gegen den er gewonnen hat.

Mehrere Ranglisten

Basic- und Premium-Abonnenten können an mehreren Ranglisten teilnehmen. Wenn man das Ranglisten-Icon drückt, erhält man eine Übersicht aller Ranglisten, an denen man teilnimmt. Wählt man aus dieser Übersicht eine Rangliste, wechselt der die Reiter Rangliste sogleich auf die gewählte Rangliste. Es ist auch möglich die Resultate der gewählten Rangliste im Reiter Resultate zu sehen.

Resultate sind nicht an Ranglisten gebunden. Für Spieler, die gleichzeitig in mehreren gleichen Rangliste vertreten sind, gelten eingetragene Resultate für alle Ranglisten, an denen beide Spieler teilnehmen. Die Punkteveränderung kann jedoch von Rangliste zu Rangliste variieren.

Öffentliche und private Ranglisten

Wenn eine Rangliste erstellt wird, kann diese entweder öffentlich oder privat konfiguriert werden. Jeder kann öffentliche Ranglisten suchen und beitreten. Um einer privaten Rangliste beizutreten, benötigt man deren Ranglisten-Code oder den individuellen Teilnehmer-Code.

Einen Ranglisten-Code teilen

Ranglisten-Administratoren können den Ranglisten-Code sehen und verteilen, indem sie in den Administrator-Modus wechseln. Dort können sie auf den "Teilen"-Button neben dem Ranglisten-Code drücken um diesen via Sozialen Medien oder E-Mail zu teilen. Eine andere Möglichkeit ist es, den Code im Verein/Club den Mitgliedern auszuhändigen.
