Slappy Login Seite

Wenn man die App das erste Mal startet, landet man auf der Login-Seite. Auf dieser Seite kann man sich bei der App anmelden, ein neues Konto kreieren oder eine neue Rangliste gründen.

---

Einer Rangliste beitreten

+++

Creating a user account

+++ Before creating an account, new users must first create a new ladder +++ or join an existing one. +++ They can join using a code that they have received from their club or another ladder member, +++ or by searching for a public ladder in their area.

Mit einem Code

Wenn ein neuer Spieler einen Ranglisten- oder Teilnehmer-Code erhalten hat, kann er diesen benutzen um der Rangliste beizutreten, indem er den "Konto eröffnen/Rangliste beitreten"-Button drückt und danach "Ranglisten-/Teilnehmer-Code eingeben" wählt. --- Dort gibt der Spieler den Code ein und drückt danach den "Find Code"-Button. +++ // Changed 'Find Code' to 'Submit Code'. +++ After entering their code, they can click the “Submit code” button. +++ Thereafter, they can complete the user registration process.

---

Ohne Code

+++

Creating a ladder or searching for a public ladder

Wenn ein Spieler eine neue Rangliste gründen möchte oder einer Rangliste beitreten möchte, ohne jeglichen Code zu kennen, kann er das indem er den "Konto eröffnen/Rangliste beitreten"-Button drückt und dann den "Suche/Gründe Rangliste"-Button wählt. Jetzt muss er sein Land und seine Region angeben um eine Liste aller verfügbarer Ranglisten in seiner Umgebung zu erhalten. --- Wenn die Rangliste, nach der er sucht, nicht existiert, kann er den "+"-Button drücken um eine neue Rangliste zu gründen. +++ If they find the ladder they are searching for, and it is public, they can join the ladder by selecting it. +++ If they find the ladder, but it is marked as private, +++ they will need to contact the club or other ladder members to request a code to join the ladder. +++ If the ladder they are looking for doesn't exist, they can press the “+” button to +++ create a new ladder. +++ If no ladders are found in a region, the player will automatically be prompted to create a new ladder.

+++

Creating a new ladder

+++ If, after searching for a ladder, a user can't find it, +++ they can create a new ladder by pressing the “+” button. +++ The user will need to enter a unique ladder name for their region, and optionally select +++ an image for the ladder and their club name.

---

Als Spieler registrieren

+++

Registering a user

--- Nachdem eine Rangliste gewählt oder gegründet wurde, muss der Spieler seinen Namen, Vornamen, E-Mail-Adresse --- und Passwort eingeben. Danach kann er sich mit der E-Mail-Adresse --- und dem gewählten Passwort bei der App anmelden. +++ After selecting a ladder to join, or creating a ladder, the user must enter their name, surname, email address +++ and password, to register themselves on the system. +++ A competitor, linked to the ladder, will automatically be created for the player when they register. +++ Thereafter, the user will be able to login to the app using the email address and password they have supplied.

Optional kann ein Spieler auch einen Avatar-Bild - idealerweise ein Portraitfoto ab Schulterhöhe - hochladen. Es wird empfohlen, dies zu tun, vor allem damit man von anderen - speziell von neuen Spielern - leichter gefunden/erkannt wird.

Anmelden

Nachdem sich ein Spieler registriert hat, kann er sich mit der angegebenen E-Mail-Adresse und dem gewählten Passwort bei der App anmelden.

Passwort erinnern

Wenn man sich einloggt, kann man wählen, dass sich die App an das Passwort erinnert. Wenn das gewählt wird, wird sich der Spieler künftig nicht mehr jedes Mal anmelden müssen, ausser wenn er sich im Hauptmenu absichtlich abmeldet.

Vergessenes Passwort wiederherstellen

Wenn ein Spieler sein Passwort vergisst, kann er es mit Hilfe seiner E-Mail-Adresse und indem er den "Passwort vergessen?"-Link drückt wiederherstellen. Nachdem der Spieler bestätigt, dass er das Passwort zurücksetzen will, erhält er eine E-Mail mit einem Bestätigungs-Code welcher er auf der "Passwort zurücksetzen"-Seite eingeben kann, um folglich ein neues Passwort zu kreieren.
