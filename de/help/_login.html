Slappy Login Seite

Wenn man die App das erste Mal startet, landet man auf der Login-Seite. Auf dieser Seite kann man sich bei der App anmelden, ein neues Konto kreieren oder eine neue Rangliste gründen.

Einer Rangliste beitreten

Mit einem Code

Wenn ein neuer Spieler einen Ranglisten- oder Teilnehmer-Code erhalten hat, kann er diesen benutzen um der Rangliste beizutreten, indem er den "Konto eröffnen/Rangliste beitreten"-Button drückt und danach "Ranglisten-/Teilnehmer-Code eingeben" wählt. Dort gibt der Spieler den Code ein und drückt danach den "Find Code"-Button.

Ohne Code

Wenn ein Spieler eine neue Rangliste gründen möchte oder einer Rangliste beitreten möchte, ohne jeglichen Code zu kennen, kann er das indem er den "Konto eröffnen/Rangliste beitreten"-Button drückt und dann den "Suche/Gründe Rangliste"-Button wählt. Jetzt muss er sein Land und seine Region angeben um eine Liste aller verfügbarer Ranglisten in seiner Umgebung zu erhalten. Wenn die Rangliste, nach der er sucht, nicht existiert, kann er den "+"-Button drücken um eine neue Rangliste zu gründen.

Als Spieler registrieren

Nachdem eine Rangliste gewählt oder gegründet wurde, muss der Spieler seinen Namen, Vornamen, E-Mail-Adresse und Passwort eingeben. Danach kann er sich mit der E-Mail-Adresse und dem gewählten Passwort bei der App anmelden.

Optional kann ein Spieler auch einen Avatar-Bild - idealerweise ein Portraitfoto ab Schulterhöhe - hochladen. Es wird empfohlen, dies zu tun, vor allem damit man von anderen - speziell von neuen Spielern - leichter gefunden/erkannt wird.

Anmelden

Nachdem sich ein Spieler registriert hat, kann er sich mit der angegebenen E-Mail-Adresse und dem gewählten Passwort bei der App anmelden.

Passwort erinnern

Wenn man sich einloggt, kann man wählen, dass sich die App an das Passwort erinnert. Wenn das gewählt wird, wird sich der Spieler künftig nicht mehr jedes Mal anmelden müssen, ausser wenn er sich im Hauptmenu absichtlich abmeldet.

Vergessenes Passwort wiederherstellen

Wenn ein Spieler sein Passwort vergisst, kann er es mit Hilfe seiner E-Mail-Adresse und indem er den "Passwort vergessen?"-Link drückt wiederherstellen. Nachdem der Spieler bestätigt, dass er das Passwort zurücksetzen will, erhält er eine E-Mail mit einem Bestätigungs-Code welcher er auf der "Passwort zurücksetzen"-Seite eingeben kann, um folglich ein neues Passwort zu kreieren.
